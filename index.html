02 / OpenSCAD Magnetic Levitation

Parametric Magnetic Levitation Motor

An axial-flux BLDC motor where the rotor levitates on opposing permanent-magnet rings — zero mechanical bearing friction. A Halbach array doubles stator-side flux without adding magnets.

Constraint
Eliminate bearing friction as the dominant loss mechanism while keeping all geometry parametric.
Signal
Halbach array, dual levitation ring pairs, 3-phase coil layout, flux-return plate, transparent shroud.
Outputs
OpenSCAD source with assembled, exploded, rotor-only, stator-only, and cross-section modes.

Process

Each project follows the same four-step design process.

01

Define the constraint

Start from fit, loading, mounting, printability, or documentation requirements instead of styling first.

02

Encode the geometry

Turn dimensions into parameters so variants are created by changing input values, not by remodeling.

03

Check the handoff

Export the right formats, preserve naming clarity, and include notes or tolerances where they matter.

04

Keep it reviewable

Use script-based CAD so changes are version-controlled, inspectable, and easier to discuss in a team setting.
